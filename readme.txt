# ⚛️ React Projects Collection

A curated collection of mini React projects built using **Vite** and **React.js** — designed for learning, practicing, and demonstrating frontend development concepts. Each folder in this repository is a standalone project with its own focus, ranging from UI interactions to small applications.

---

## 📁 Projects Overview

| Project Folder       | Description                                                                 |
|----------------------|-----------------------------------------------------------------------------|
| `01vitereact`         | ⚙️ Initial React + Vite setup (boilerplate for quick project bootstrapping) |
| `02counter`           | 🔢 A basic counter app using `useState`                                     |
| `04bg-changer`        | 🎨 Dynamic background color changer                                          |
| `05passwordgenerator` | 🔐 Password generator with adjustable options (length, characters, etc.)     |
| `customReactt`        | 🧪 React rendering made from javascript             |
| `cvresume`            | 🧾 Resume UI using React — a practice for component layout and structure     |
| `pokemonmem`          | 🧠 Pokémon memory game using cards & matching logic                          |

---

## 🛠️ Tech Stack

- ⚛️ **React.js**
- ⚡ **Vite** – Fast dev server and bundler
- 🎨 **CSS3** (or Tailwind if added)
- 🧪 **JavaScript ES6+** features
